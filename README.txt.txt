Random Password Generator

Автор: [Ищейкина Екатерина Алексеевна]

Описание:Графическое приложение для генерации случайных
паролей с возможностью настройки длины и используемых символов 
(цифры, буквы, спецсимволы). История созданных паролей сохраняется 
в файл history.json.

Возможности:

Настройка длины пароля с помощью ползунка.
Выбор типов символов через чекбоксы.
Сохранение истории сгенерированных паролей.
Как запустить:

Убедитесь, что установлен Python 3.
Запустите файл password_generator.py.
Технологии:

Язык: Python 3.
Библиотека для интерфейса: tkinter.
